🎰 Slot Machine Game

A simple Python-based terminal slot machine game.

🚀 How to Play

1. Deposit money to start.

2. Choose the number of lines to bet (1-3).

3. Place a bet per line.

4. Spin and check if you win!

🏆 Winning Rules

Match symbols in a line to win.

Symbol values:

A = 5x bet

B = 4x bet

C = 3x bet

D = 2x bet
